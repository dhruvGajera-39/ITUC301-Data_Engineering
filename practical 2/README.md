# Practical 2 – Data Ingestion and ETL Pipeline

## 📖 Overview

This repository contains the implementation of **Practical 2** for the **ITUC301 – Data Engineering** course.

The practical demonstrates the process of collecting data from different sources, performing data validation and transformation, and loading the processed data into a database as part of a basic ETL (Extract, Transform, Load) pipeline.

---

## 🎯 Objective

- To understand the ETL (Extract, Transform, Load) process.
- To ingest data from various sources.
- To clean and validate the collected data.
- To store processed data in a relational database.
- To generate logs for monitoring and debugging.

```bash

```

3. Install the required libraries.

```bash

```

4. Run the program.

